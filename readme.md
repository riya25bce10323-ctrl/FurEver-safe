🐾 FurEver Safe
FurEver Safe is a web-based animal welfare application built to support animal rescue, adoption, and NGO connectivity.
The platform enables users to report animals in need, explore adoption opportunities, and connect with trusted NGOs — all through a simple, responsive interface.
🌟 Features
🐶 Rescue Module
Report animals in need of rescue
Capture animal type and location
View all rescue requests in a clean list format
🏠 Adoption Module
Add animals available for adoption
View a list of adoptable animals
Simple and user-friendly adoption flow
🤝 NGO Connectivity
Dedicated NGO contact page
Direct call/email links to animal welfare NGOs
Designed for quick real-world action
🎨 UI & UX
Clean, modern UI with soft green theme
Card-based layout
Fully mobile responsive
Consistent design across all pages
🛠️ Tech Stack
Frontend: HTML, CSS (Flexbox, Responsive Design)
Backend: Python, Flask
Version Control: Git & GitHub
Development Tool: VS Code
